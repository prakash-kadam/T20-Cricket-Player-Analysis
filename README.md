# T20 Cricket Player Performance Dashboard

## Project Overview
This Power BI dashboard analyzes T20 cricket player performance using various KPIs and visualizations.

## Tools Used
- Power BI
- Excel
- Data Analysis
- Data Visualization

## Key Metrics
- Total Players
- Total Runs
- Total Fifties
- Total Centuries
- Batting Average

## Dashboard Features
- Runs by Player
- Players by Country
- Sixes by Player
- Runs vs Strike Rate Analysis
- Country Filter

- Power BI Dashboard (.pbix)
- Project Report
- Dataset
- Dashboard Screenshots
